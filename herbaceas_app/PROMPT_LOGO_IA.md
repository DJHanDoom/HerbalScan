# Prompt para Geração de Logo - HerbalScan

## Prompt Principal (DALL-E / Midjourney / Stable Diffusion)

```
Create a professional, modern logo for "HerbalScan", a scientific software application for analyzing herbaceous vegetation coverage using artificial intelligence. 

The logo should combine:
- A stylized representation of herbaceous plants or grass (delicate leaves, stems)
- A subtle grid or square pattern suggesting scientific measurement (1x1 meter analysis squares)
- A technological/analytical element (could be a scan line, data visualization, or AI symbol like a neural network node or hexagon)
- Clean, professional design suitable for a Windows desktop application icon

Style: Modern, minimalist, scientific, professional. Flat design with subtle gradients. 
Color palette: Deep forest greens (#0d3d1f, #1a5f2e), bright accent greens (#22c55e, #4ade80), and a tech blue accent (#3b82f6, #60a5fa).
Background: Circular or rounded square, dark green gradient.
Foreground: Light green/white plant elements with blue tech accent.

The logo should be:
- Scalable from 16x16 to 512x512 pixels
- Clear and recognizable at small sizes
- Professional and scientific in appearance
- Not too busy or cluttered
- Suitable for use as a Windows application icon

Format: Square aspect ratio, suitable for icon use. Clean edges, no text (icon only).
```

## Especificações Técnicas para a IA

### Aspectos Importantes:
1. **Formato**: Quadrado (1:1), adequado para ícone
2. **Fundo**: Circular ou quadrado arredondado com gradiente verde escuro
3. **Elementos principais**:
   - Planta herbácea estilizada (folhas delicadas, caules)
   - Grid sutil de medição (representando quadrados de 1x1m)
   - Indicador de tecnologia (hexágono, linha de scan, ou nó de rede neural)
4. **Cores**:
   - Fundo: Verde floresta escuro (#0d3d1f, #1a5f2e)
   - Vegetação: Verde claro/brilhante (#22c55e, #4ade80, #86efac)
   - Tecnologia: Azul (#3b82f6, #60a5fa)
5. **Estilo**: Minimalista, profissional, científico, moderno
6. **Sem texto**: Apenas elementos visuais (o nome será adicionado separadamente se necessário)

### O que EVITAR:
- ❌ Texto ou letras no logo
- ❌ Design muito complexo ou carregado
- ❌ Cores muito vibrantes ou chamativas
- ❌ Estilo muito infantil ou casual
- ❌ Elementos que não sejam reconhecíveis em tamanhos pequenos

### O que INCLUIR:
- ✅ Design limpo e profissional
- ✅ Elementos que funcionem bem em tamanhos pequenos (16x16px)
- ✅ Paleta de cores científica e profissional
- ✅ Combinação harmoniosa de natureza + tecnologia
- ✅ Formato adequado para ícone de aplicativo Windows

   - Leonardo.ai
   - Outras IAs de imagem

3. **Gere múltiplas variações** e escolha a melhor
4. **Ajuste o prompt** se necessário baseado nos resultados
5. **Exporte em alta resolução** (512x512 ou maior)
6. **Use o script `create_icon.py`** para converter para os formatos necessários (PNG, ICO, SVG)

## Dicas de Refinamento

Se o resultado inicial não estiver perfeito, adicione ao prompt:
- "more minimalist" - para um design mais simples
- "more scientific" - para um aspecto mais técnico
- "softer colors" - para cores mais suaves
- "stronger contrast" - para melhor visibilidade
- "icon optimized" - para garantir que funciona bem em tamanhos pequenos
